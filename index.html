Admin Control Center

Edit the app from here. Save Preview updates this device immediately. Download Publish File creates the new data.js file to upload to GitHub so every family sees the update.

How to Publish Updates

  1. Edit the content below.
  2. Tap Save Preview and review the app on this device.
  3. Tap Download Publish File.
  4. In GitHub, replace only data.js.
  5. Commit changes and refresh the app with ?v=14.

Most weekly updates should only require replacing data.js. Do not edit layout files unless we are redesigning the app.
